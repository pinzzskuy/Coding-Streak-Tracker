# Coding-Streak-Tracker
🚀 Daily Coding Logger: Absen dan tambahkan +1 Streak, cek durasi coding hari ini dan kemarin.

🔥 Daily Coding Streak Tracker

Program sederhana untuk ngetrack konsistensi koding kamu!

✨ Fitur
- ✅ Absen Harian: Koding setiap hari dan agar mau tahu udah berapa koding, tinggal run program, masukkan dan otomatis sistem menmbah 1+
- ⏱️ Time Tracking: Masukkan input dan waktu/sesi koding lu hari ini terekam.
- 📊 Daily Report: Sistem akan memberi tahu sesi kemarin koding berapa jam dan udah berapa hari lo konsisten belajar coding.
- 🚀 Streak Counter:Pastikan motivasi terjaga dengan streak tetap ijo.

⚡ Motivasi
"Consistency is key to becoming a better developer."

Alasan
Kenapa saya membuat ini? karena saya sendiri kadang merasa malas walaupun belum membuka file program, karena itu saya sendiri membuat program streak coding untuk projek kecil saya dan sebagai saya belajar PYTHON agar saya konsisten dalam belajar coding setiap hari, seperti streak tiktok dan duolingo, streak ini akan membuat otak kita kecanduan untuk koding hari ini dan hari selanjutnya untuk mencapai jumlah streak tertentu, sehingga dapat membangun sebuah habit kecil coding yang mempermudah jika kita ingin menekuni bidang programming.
